# steel/identity
### **rational?**
Returns #t if obj is a rational number, #f otherwise.
Rational numbers are numbers that can be expressed as the quotient of two numbers.
For example, 3/4, -5/2, 0.25, and 0 are rational numbers, while

(rational? value) -> bool?

Examples:
```scheme
(rational? (/ 0.0)) ⇒ #f
(rational? 3.5)     ⇒ #t
(rational? 6/10)    ⇒ #t
(rational? 6/3)     ⇒ #t
```
